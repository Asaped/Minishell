# Minishell
[Aceyzz](https://github.com/aceyzz) & 
[Asaped](https://github.com/Asaped)

## Introduction
Minishell est notre interprétation simplifiée d'un shell Unix, développé dans le cadre de notre projet à l'école 42. Ce projet nous permet de comprendre les fondamentaux de la programmation système sous Unix.

## Installation
1. Cloner le dépôt : `git clone https://github.com/Asaped/Minishell`
2. Naviguer dans le répertoire du projet : `cd Minishell/minishell`
3. Compiler le programme : `make`

## Utilisation
Pour lancer Minishell :
```bash
./minishell
```

Une fois Minishell lancé, vous pouvez :

Exécuter des commandes standard Unix.
Gérer les redirections et les pipes.
Utiliser les variables d'environnement.

## Fonctionnalités

- Interprétation des commandes.
- Gestion des signaux.
- Expansion des variables.
- Redirections et pipes.

## Quitter Minishell

Pour quitter le programme, utilisez la commande :
```bash
exit // ou utiliser la commande CTRL+D
```
